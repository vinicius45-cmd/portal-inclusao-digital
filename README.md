# Portal de Inclusão Digital - Riacho Fundo 2

## Sobre o Projeto

Portal criado como parte de um projeto de extensão universitária com o objetivo de promover a inclusão digital na comunidade do Riacho Fundo 2, Brasília - DF.

## Acesse o Site

[https://vinicius45-cmd.github.io/portal-inclusao-digital/](https://vinicius45-cmd.github.io/portal-inclusao-digital/)

## Objetivo

Fornecer tutoriais gratuitos e acessíveis sobre:
- Serviços digitais do Gov.br
- WhatsApp Business
- Controle financeiro
- Proteção contra golpes digitais
- Consultas públicas online
- E muito mais!

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Licença

Este projeto é de código aberto e está disponível para fins educacionais.
